# BATYAG-Sentiment-Analysis-AI
BATYAG AI: A highly calibrated sentiment analysis web application powered by TensorFlow. Named after the Hiligaynon word for "feeling" or "sense," it utilizes a Bidirectional LSTM and GloVe transfer learning to decode the polarity of human text.   sentiment-analysis tensorflow keras nlp  glove-embeddings temperature-scaling model-calibration
